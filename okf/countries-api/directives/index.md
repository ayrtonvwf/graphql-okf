# Directives

- [deprecated](deprecated.md) — Marks an element of a GraphQL schema as no longer supported.
- [include](include.md) — Directs the executor to include this field or fragment only when the `if` argument is true.
- [oneOf](oneOf.md) — Indicates exactly one field must be supplied and this field must not be `null`.
- [skip](skip.md) — Directs the executor to skip this field or fragment when the `if` argument is true.
- [specifiedBy](specifiedBy.md) — Exposes a URL that specifies the behavior of this scalar.
